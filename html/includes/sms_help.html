Hilfe zum SMS-Versand

Der SMS-Versand mithilfe des SMS-Systems von Holy-Wars 2 (kurs HW2SMS genannt) ist kinderleicht.

Senden von SMS an Spieler

Im Feld Empfänger tragen Sie den Namen des Spielers ein, dem Sie eine SMS senden möchten. Der Spieler muss eine SMS-Empfangsnummer unter My HW2 - SMS-Einstellungen hinterlegt haben.
Ob ein Spieler eine SMS-Empfangsnummer hinterlegt hat, erfahren Sie in der Spielerinfo. Dann erscheint nämlich das HW2SMS-Logo.

Senden von SMS an deutsche Mobilfunknummern

Sie können eine SMS direkt an eine bekannte Nummer senden. Dazu müssen Sie allerdings das vollständige internationale Format der Nummer eingeben. Wollen Se z.B. eine SMS an die Mobilfunknummern 0171-78912345 senden, dann müssen Sie +4917178912345 eingeben. Bitte beachten Sie, dass die 0 der Handy-Vorwahl weggelassen werden muß.
Als Ländervorwahl ist im Moment lediglich +49 für Deutschland erlaubt. Auch wenn Sie nicht in Deutschland wohnen können Sie HW2SMS an deutsche Mobilfunknummern versenden.
